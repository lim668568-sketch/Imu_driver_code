# IMU姿态传感器上位机软件

## 项目简介

这是一个基于Qt开发的IMU姿态传感器数据采集与可视化软件，支持MPU6500、MPU6050、ICM42688等常用姿态传感器。

## 主要功能

### 1. 串口通信

- 自动扫描可用串口
- 支持多种波特率(9600-921600)
- 自定义协议解析
- 数据缓存与丢包处理

### 2. 三维姿态显示

- **3D模型旋转**: 实时显示姿态变化
- **多种模型**: 飞机、汽车、手机、立方体
- **鼠标交互**: 支持旋转、缩放视角
- **姿态仪表盘**: 地平线、罗盘、角度指示

### 3. 数据可视化

- **实时波形图**: 六轴数据+欧拉角曲线
- **时间窗口**: 5s/10s/30s/60s可调
- **多通道显示**: 支持同时显示多个数据通道
- **鼠标测量**: 显示鼠标位置的时间和数值

### 4. 数据分析

- **统计分析**: 均值、方差、最大最小值
- **FFT频谱分析**: 实时频谱显示
- **噪声分析**: 噪声密度计算

### 5. 数据管理

- **数据记录**: 自动保存为CSV
- **数据导出**: 导出为CSV格式
- **数据回放**: 导入历史数据回放

## 编译步骤

### 使用Qt Creator

1. 打开 `IMU_Uploader.pro` 文件
2. 配置编译套件(Kit)
3. 点击"构建" -> "构建项目"
4. 点击"运行"

### 使用命令行

```bash
cd IMU_Uploader
qmake
make
```

### Windows下生成exe

1. 编译Release版本
2. 使用 `windeployqt` 工具:

```bash
windeployqt IMU_Uploader.exe
```

## 使用说明

### 1. 连接设备

1. 选择正确的串口号
2. 设置波特率(默认115200)
3. 点击"连接"按钮

### 2. 数据协议

默认协议格式:

```
帧头(2B) + 帧长(1B) + 数据类型(1B) + 数据(NB) + 校验(1B) + 帧尾(2B)
示例: [0x55 0xAA] [0x0D] [0x01] [Roll(4B) Pitch(4B) Yaw(4B)] [CRC] [0x0D 0x0A]
```

### 3. 数据显示

- **实时监控**: 3D模型+姿态仪表盘
- **波形显示**: 数据曲线图
- **数据记录**: 自动保存

### 4. 快捷操作

- **清空数据**: 工具栏"清空数据"按钮
- **暂停显示**: 工具栏"暂停"按钮
- **导出数据**: 菜单"文件" -> "导出数据"
- **重置视图**: 菜单"视图" -> "重置视图"

## 配置文件

### config.ini

程序会自动创建配置文件，保存:

- 串口设置
- 窗口布局
- 波形参数

### 日志文件

日志保存在 `logs/` 目录下。

## 扩展开发

### 添加新协议

1. 在 `ProtocolParser.cpp` 中添加解析逻辑
2. 或在 `protocols/` 目录添加JSON配置文件

### 添加新模型

1. 在 `ThreeDView.cpp` 的 `drawModel()` 中添加
2. 实现 `drawYourModel()` 函数

### 添加新分析功能

1. 在 `src/data/` 中添加新类
2. 在主窗口中集成

## 已知问题

1. OpenGL在某些集成显卡上可能需要更新驱动
2. 高采样率时建议降低波形更新频率
